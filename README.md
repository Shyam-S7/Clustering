# Customer Segmentation using K-Means Clustering

## Overview
This project segments customers into different groups based on their demographics and purchasing behavior using K-Means Clustering.

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Scaling
- K-Means Clustering
- Elbow Method
- Cluster Visualization

## Dataset
The dataset contains:
- Age
- Gender
- Income
- Spending Score
- Membership Years
- Purchase Frequency
- Preferred Category
- Last Purchase Amount

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Workflow
1. Load Dataset
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Scaling
5. K-Means Clustering
6. Elbow Method
7. Cluster Analysis
8. Visualization

## Results
Successfully identified customer segments based on purchasing patterns and demographics.
